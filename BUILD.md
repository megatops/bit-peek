# How To Build

1. Install `vsce`:

   ```bash
   npm install -g vsce
   ```

2. Build the extension package:

   ```bash
   vsce package
   ```

3. Compile only:

   ```bash
   npm run compile
   ```

4. Compile and run tests:

   ```bash
   npm run test
   ```

5. Publish to the Marketplace:

   ```bash
   vsce publish
   ```
